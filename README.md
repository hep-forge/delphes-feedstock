Feedstock for [delphes](https://github.com/delphes/delphes) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Delphes: fast multipurpose detector response simulation

## Install

```bash
conda install -c hep-forge -c conda-forge delphes
```
